# Análise de Falhas de Manutenção

Projeto de análise de dados operacionais com foco na identificação de padrões de falhas em equipamentos e apoio à tomada de decisão.

## Objetivo
Identificar falhas recorrentes, tempo médio de parada e oportunidades de melhoria de processos.

## Ferramentas
- Python
- Pandas
- Matplotlib
- Excel

## Etapas
- Limpeza de dados
- Análise exploratória
- Visualização de dados
- Geração de insights
